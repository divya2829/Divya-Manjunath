# Event Management Website 🎉

## 📌 Description
A Django-based event management platform that allows users to browse themes, book events, and enables admins to manage venues and bookings.

## 🚀 Features
- User authentication (Login/Register)
- Event booking system
- Admin dashboard for managing events
- Theme and venue selection
- Responsive UI

## 🛠 Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- SQLite

## 📷 Future Improvements
- Payment integration
- Live deployment
- Email notifications
